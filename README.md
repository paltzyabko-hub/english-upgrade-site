# English Upgrade Site

Free static website for **English Upgrade: From 4 Units to 5 Units**.

## How to publish on GitHub Pages
1. Create a new GitHub repository named `english-upgrade-site`.
2. Upload all files from this folder.
3. Go to repository Settings → Pages.
4. Choose branch `main` and folder `/root`.
5. Your free site will be available at:
   `https://YOUR-GITHUB-USERNAME.github.io/english-upgrade-site/`

Later, QR codes in the workbook can point to the pages inside `/unit1/`.
